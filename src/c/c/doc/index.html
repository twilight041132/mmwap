WAP调起

  • WAP调起DEMO(应用详情展示浏览器名称)
  • WAP调起DEMO(应用详情不展示浏览器名称)


WAP调起客户端API

步骤说明代码示例
1.导入JS 
导入MM WAP调起JavaScript代码

<script type="text/javascript" src="http://a.10086.cn/c/mmapp.js" charset="utf-8" ></script>
2.初始化配置 
调用MM客户端展示指定应用详情页面，可通过showtitle配置MM详情页标题栏显示的浏览器名称，showtitle设置为false表示不显示，不设置或者设置为true表示显示；
通过channelid配置渠道ID，配置的channelid需要保证有对应的MM包下载，不配置的时候为默认渠道ID（5410093632）。代码在加载mmapp.js后执行;


<script>  	
mm.init({
	showtitle:是否显示,
	channelid:渠道ID 
})
<script>
3.应用下载 
按钮添加触发事件mm.download(应用ID)，调用MM客户端下载指定应用

<a onclick="mm.download(应用ID)" >下载应用</a>
3.应用详情 
按钮添加触发事件mm.detail(应用ID)，调用MM客户端展示指定应用详情页面

<a onclick="mm.detail(应用ID)" >应用详情</a>
