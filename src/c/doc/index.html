WAP调起客户端API

步骤说明代码示例
1.导入JS 
导入MM WAP调起JavaScript代码

<script type="text/javascript" src="http://a.10086.cn/c/mmapp.js" charset="utf-8" ></script>
2.初始化配置 
提供各种参数配置，可通过mm.init/mm.set方法配置，代码在加载mmapp.js后执行，
详细配置参数，可查看WAP调起API提供的参数配置说明;


<script>  	
mm.init({
	showtitle:true,//true：显示；false：隐藏
	channelid:5410093632 //a.l0086.cn 默认使用渠道ID：5410093632
})
<script>
3.应用下载 
按钮添加触发事件mm.download(应用ID)，调用MM客户端下载指定应用

<a onclick="mm.download(应用ID)" >下载应用</a>
3.应用详情 
按钮添加触发事件mm.detail(应用ID)，调用MM客户端展示指定应用详情页面

<a onclick="mm.detail(应用ID)" >应用详情</a>
3.应用批量下载 
js调用mm.batchDownload(应用ID列表)触发应用批量下载,应用ID列表格式可以是字符串或数组。

字符串格式：使用"/"分割列表ID，eg:"应用ID1/应用ID2/..."

数组格式：把应用ID列表按数组存储即可，eg:[应用ID1,应用ID2,...]


mm.batchDownload("/300002784578/300002457498")

WAP调起API提供的参数配置说明

可配置参数：

batchMaxApps:"15" ;//批量下载最大应用数，默认15，目前客户端最多支持15个应用下载
showtitle:true ;//调起客户端，在客户端显示调起端浏览器名字，默认显示
channelid:'5410093632';//渠道ID设置，a.l0086.cn 默认使用渠道ID：5410093632
onIntent:true;//端口激活客户端失败，默认启用intent调起
reCall:2000;//端口激活客户端失败,启用intent调起之后，默认2秒后执行用户操作
lockTime:2000;//intent调起的时间锁，默认2秒内只能进行一次intent调起

参数配置方法，不配置都是用默认值：

1.通过初始化配置

mm.init({
	showtitle:true,
	channelid:'5410093632',
	onIntent:true,
	reCall:2000,
	lockTime:2000
})

2.通过set方法配置

mm.set("参数名",参数值);
举例，设置channelid:mm.set("channelid",'5410093632');
